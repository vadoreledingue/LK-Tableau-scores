## Site web interactif - Tableau des scores

Base du projet pour suivre les scores de **5 equipes** avec **3 categories** et un suivi par saison.

### Fichiers
- `index.html` : structure de la page
- `styles.css` : theme bleu roi / jaune or + responsive
- `script.js` : logique interactive (ajout, filtres, classement, stockage local)

### Fonctionnalites
- Ajout de plusieurs resultats
- Saisie obligatoire de la saison (format `YYYY-YYYY`, ex: `2025-2026`)
- Classement global filtrable par saison
- Historique filtrable (saison/equipe/categorie)
- Donnees sauvegardees dans `localStorage`

### Lancer le projet
1. Ouvrir `index.html` dans un navigateur.
2. Ajouter les resultats au fur et a mesure de la saison.
